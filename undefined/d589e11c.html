文件后缀名大全：分类、用途与详解

文件后缀名（File Extension），通常是文件名中最后一个点号（.）之后的部分，它提示操作系统和用户文件的类型以及可能用于打开它的应用程序。了解常见的文件后缀名有助于我们更好地管理和使用数字文件。

下面是一份较为全面的文件后缀名列表，按常见用途分类：

📄 办公文档、电子书、标记语言及纯文本

这类文件通常用于记录信息、写作、排版和阅读。

后缀名 (Extension) 文件类型 常见用途/说明
.docx Word 文档 Microsoft Word (2007+)
.doc Word 文档 Microsoft Word (97-2003)
.pdf PDF 文档 便携式文档格式，跨平台阅读，保留格式
.txt 纯文本 无格式文本，通用性极高
.rtf 富文本格式 带基本格式的文本，跨平台兼容性较好
.odt ODF 文本文档 OpenOffice/LibreOffice Writer 文档
.pages Pages 文档 Apple Pages 文档
.wps WPS Office 文档 金山 WPS Office 文档格式
.tex LaTeX 源文件 科学排版系统源文件
.md Markdown 文件 轻量级标记语言，常用于 README、笔记、网页内容
.epub EPUB 电子书 开放标准的电子书格式，可重排版
.mobi Mobipocket 电子书 早期 Kindle 支持的格式
.azw / .azw3 Kindle 电子书 亚马逊 Kindle 专有电子书格式
.indd InDesign 文档 Adobe InDesign 专业排版设计文件

🖼️ 图像文件

包含从简单的位图到复杂的矢量图和专业摄影格式。

后缀名 (Extension) 文件类型 常见用途/说明
.jpg / .jpeg / .jfif JPEG 图像 最常见的有损压缩图像格式，适合照片
.png PNG 图像 无损压缩图像格式，支持透明背景
.gif GIF 图像 支持动画和透明，颜色数有限 (256色)
.bmp 位图图像 无压缩或很少压缩的图像，文件较大
.tiff / .tif TIFF 图像 高质量图像格式，支持无损压缩和多页，常用于印刷
.webp WebP 图像 Google 开发的现代图像格式，提供有损和无损压缩
.svg SVG 矢量图 基于 XML 的可缩放矢量图形，放大不失真
.ico 图标文件 Windows 图标文件
.heif / .heic HEIF 图像 高效图像文件格式，常用于苹果设备
.avif AVIF 图像 基于 AV1 视频编码的新一代高效图像格式
.psd Photoshop 文档 Adobe Photoshop 原生文件，包含图层等编辑信息
.ai Illustrator 文件 Adobe Illustrator 原生矢量图形文件
.dng DNG 数字负片 Adobe 推出的开放式 RAW 图像格式

🎥 视频文件

存储动态影像和声音的各种格式。

后缀名 (Extension) 文件类型 常见用途/说明
.mp4 / .m4v MPEG-4 视频 最流行的视频容器格式，兼容性好
.avi AVI 视频 早期的 Windows 视频容器格式
.mov QuickTime 视频 Apple QuickTime 格式，常用于视频编辑
.mkv Matroska 视频 开源的视频容器格式，支持多音轨、多字幕
.flv Flash 视频 曾用于网络流媒体，现已较少使用
.wmv Windows Media 视频 Microsoft 开发的视频格式
.webm WebM 视频 开放、免费的 Web 视频格式，常用于 HTML5
.mpg / .mpeg MPEG 视频 较早的视频压缩标准格式
.vob DVD 视频对象 DVD 光盘中的主要视频文件格式
.ogv Ogg Vorbis 视频 Ogg 项目的视频格式
.3gp 3GPP 多媒体文件 主要用于 3G 移动电话的多媒体容器格式
.ts / .mts / .m2ts MPEG 传输流 常用于数字广播 (DVB, ATSC) 和蓝光/AVCHD 光盘
.aep After Effects 项目 Adobe After Effects 视频特效与合成项目文件

🎵 音频文件

存储声音信息的各种编码和格式。

后缀名 (Extension) 文件类型 常见用途/说明
.mp3 MP3 音频 最流行的有损压缩音频格式
.wav WAV 音频 无压缩或很少压缩的音频格式，常用于 Windows 和专业音频
.ogg Ogg Vorbis 音频 开放、免费的有损压缩音频格式
.flac FLAC 音频 流行的无损压缩音频格式
.aac AAC 音频 高级音频编码，常用于 Apple 设备和流媒体
.m4a M4A 音频 通常使用 AAC 编码的 MPEG-4 音频文件，Apple 常用
.wma Windows Media 音频 Microsoft 开发的音频格式
.mid / .midi MIDI 文件 存储音符和控制信息，而非实际声音波形，文件很小
.opus Opus 音频 现代高效的开源音频编解码器，适用于语音和音乐
.aiff / .aif AIFF 音频 Apple 开发的无压缩音频格式，类似 WAV
.ape Monkey’s Audio 一种无损音频压缩格式
.amr AMR 音频 自适应多速率音频编码，常用于移动设备语音录音

📦 压缩与镜像

用于归档、减小文件大小或创建磁盘副本。

后缀名 (Extension) 文件类型 常见用途/说明
.zip ZIP 压缩包 最常见的压缩格式，跨平台支持
.rar RAR 压缩包 压缩率较高的专有格式，需要 WinRAR 或兼容软件
.7z 7z 压缩包 高压缩率的开源格式，需要 7-Zip 或兼容软件
.tar TAR 归档文件 将多个文件打包成一个文件，本身不压缩 (Unix/Linux 常用)
.gz Gzip 压缩文件 使用 Gzip 压缩的文件，常与 .tar 结合 (tar.gz)
.tar.gz / .tgz Gzip 压缩的 TAR 先打包后压缩的常见格式
.bz2 Bzip2 压缩文件 使用 Bzip2 压缩，压缩率通常比 Gzip 高
.tar.bz2 / .tbz2 Bzip2 压缩的 TAR 先打包后使用 Bzip2 压缩
.xz XZ 压缩文件 使用 XZ 压缩，压缩率通常比 Bzip2 更高
.tar.xz / .txz XZ 压缩的 TAR 先打包后使用 XZ 压缩
.cab Cabinet 文件 Microsoft Windows 压缩归档格式，常用于软件安装
.iso ISO 光盘镜像 CD/DVD/BD 的标准镜像格式
.img 磁盘镜像 通用磁盘镜像格式，可用于光盘、软盘、硬盘等
.dmg Apple 磁盘镜像 macOS 软件分发和磁盘镜像格式
.vhd / .vhdx 虚拟硬盘 Microsoft Hyper-V 和 Virtual PC 虚拟机磁盘格式
.vmdk VM 虚拟磁盘 VMware 虚拟机磁盘格式
.ova / .ovf 开放虚拟化格式 用于打包和分发虚拟机的标准格式
.wim Windows 映像格式 Windows 安装和部署中使用的基于文件的映像格式
.deb Debian 软件包 Debian, Ubuntu 等 Linux 发行版的软件包格式
.rpm RPM 软件包 Red Hat, Fedora, CentOS 等 Linux 发行版的软件包格式

💻 系统文件

操作系统运行、配置和维护所需的文件。

后缀名 (Extension) 文件类型 常见用途/说明
.dll 动态链接库 Windows 程序共享的代码库
.sys 系统文件/驱动程序 Windows 核心系统文件或设备驱动
.drv 设备驱动程序 较旧的 Windows 设备驱动
.ini 初始化配置文件 简单的文本配置文件格式，常用于 Windows 程序
.cfg / .conf 配置文件 通用配置文件后缀
.log 日志文件 记录程序或系统运行事件的文本文件
.bak 备份文件 通用备份文件后缀，也常用于 SQL Server 数据库备份
.tmp / .temp 临时文件 程序运行时产生的临时数据
.dat 数据文件 通用数据文件，格式不确定，取决于创建它的程序
.inf 安装信息文件 Windows 用于安装驱动程序或软件组件的信息文件
.url Internet 快捷方式 指向网页地址的快捷方式
.lnk Windows 快捷方式 指向本地文件、文件夹或程序的快捷方式
.cpl 控制面板项 Windows 控制面板小程序
.scr 屏幕保护程序 Windows 屏幕保护程序文件 (实际是可执行文件)
.rom ROM 映像 只读存储器映像，常用于 BIOS/固件或游戏模拟器
.nfo 信息文件 “Information”，常包含软件或发布信息的文本文件
.plist 属性列表 macOS 和 iOS 中常用的 XML 或二进制配置文件格式

⚙️ 可执行文件与脚本

可以直接运行或通过解释器执行的文件。

后缀名 (Extension) 文件类型 常见用途/说明
.exe Windows 可执行文件 Windows 应用程序主程序文件
.msi Windows 安装包 Windows Installer 数据库，用于安装、卸载软件
.bat / .cmd Windows 批处理脚本 在 Windows 命令提示符下执行的命令脚本
.sh Shell 脚本 在 Unix/Linux/macOS 终端下执行的命令脚本
.com MS-DOS 程序 早期的小型可执行文件格式
.apk Android 应用包 Android 操作系统的应用程序安装包
.ipa iOS 应用包 iOS 操作系统的应用程序安装包 (App Store Package)
.app macOS 应用程序包 macOS 应用程序 (实际是一个包含资源的文件夹)
.jar Java 归档文件 Java 平台的可执行文件或库文件
.py Python 脚本 Python 语言源代码文件，需要 Python 解释器运行
.js JavaScript 文件 主要用于网页前端交互，也可用于后端 (Node.js) 等
.vbs / .vbe VBScript 脚本 Windows 脚本宿主支持的脚本语言
.ps1 PowerShell 脚本 Windows PowerShell 脚本文件
.rb Ruby 脚本 Ruby 语言源代码文件
.pl Perl 脚本 Perl 语言源代码文件
.command macOS 终端脚本 macOS 下双击即可在终端运行的脚本文件
.action macOS Automator 工作流 macOS 自动化工具 Automator 创建的工作流文件
.wasm WebAssembly 模块 可在现代 Web 浏览器中运行的低级二进制代码格式

🌐 网页相关

构成网站和 Web 应用的文件。

后缀名 (Extension) 文件类型 常见用途/说明
.html / .htm HTML 文件 网页的标准标记语言
.xhtml XHTML 文件 基于 XML 的更严格的 HTML 版本
.css CSS 样式表 定义网页的样式和布局
.js JavaScript 文件 实现网页交互和动态功能 (见可执行文件部分)
.mjs / .cjs JavaScript 模块 ES 模块 (.mjs) 和 CommonJS 模块 (.cjs)
.ts TypeScript 文件 JavaScript 的超集，增加了静态类型，需编译成 JS
.jsx / .tsx JSX/TSX 文件 React 等框架中使用的 JavaScript/TypeScript 扩展语法
.php PHP 脚本 服务器端脚本语言，用于动态网页开发
.asp / .aspx ASP/ASP.NET 页面 Microsoft 的服务器端网页技术
.jsp JavaServer Pages 基于 Java 的服务器端网页技术
.xml XML 文件 可扩展标记语言，用于存储和传输结构化数据
.json JSON 文件 JavaScript 对象表示法，轻量级的数据交换格式
.vue Vue.js 组件文件 Vue.js 框架的单文件组件
.sass / .scss Sass/SCSS 样式表 CSS 预处理器，提供变量、嵌套等高级功能
.less Less 样式表 另一种 CSS 预处理器

.rss / .atom RSS/Atom Feed 文件 用于内容聚合和订阅的 XML 格式

📊 数据文件

存储结构化或半结构化数据。

后缀名 (Extension) 文件类型 常见用途/说明
.csv CSV 文件 逗号分隔值，纯文本表格数据，通用性好
.tsv TSV 文件 制表符分隔值，类似 CSV
.json JSON 数据文件 (见网页相关) 轻量级数据交换格式
.xml XML 数据文件 (见网页相关) 结构化数据标记语言
.db 数据库文件 通用数据库文件后缀，具体格式多样
.sqlite / .db3 SQLite 数据库 轻量级嵌入式关系数据库文件
.sql SQL 脚本 包含 SQL 语句的文本文件，用于操作数据库
.mdb / .accdb Access 数据库 Microsoft Access 数据库文件 (旧版 .mdb, 新版 .accdb)
.sdf SQL Server Compact DB SQL Server Compact 版数据库文件
.ics / .ical iCalendar 文件 日历事件和任务的标准格式
.vcf / .vcard vCard 文件 电子名片，用于存储联系人信息

.gpx GPX 文件 GPS 数据交换格式，用于记录航点、轨迹、路线

.cfg / .conf 配置文件 (见系统文件) 通用配置后缀
.yml / .yaml YAML 文件 YAML Ain’t Markup Language，易读的数据序列化格式
.toml TOML 文件 Tom’s Obvious, Minimal Language，简单的配置文件格式
.json JSON 配置文件 使用 JSON 格式存储配置
.xml XML 配置文件 使用 XML 格式存储配置
.plist 属性列表 (见系统文件) macOS/iOS 常用配置格式
.env 环境变量文件 常用于存储环境变量配置的纯文本文件
.properties Java Properties 文件 Java 平台常用的键值对配置文件格式
.reg Windows 注册表文件 包含 Windows 注册表项修改信息的文本文件

🔧 其他格式

包含各种特定用途的文件。

后缀名 (Extension) 文件类型 常见用途/说明
.kdbx KeePass 密码数据库 KeePass 密码管理器使用的加密数据库文件
.eml 电子邮件文件 单个电子邮件的标准格式，可被多种邮件客户端打开

.pak 包文件 通用后缀，常用于游戏资源打包

注意:

  • 同一个后缀名可能被不同类型的软件用于不同的目的（例如 .dat, .bak, .key, .sav）。文件的实际类型需要根据上下文或使用特定工具来判断。
  • 有些文件类型并没有标准后缀名（例如 Linux 下的可执行文件）。
